# Pasos para descargar RemoteJobs

Si se descarga por HTTP hacer git clone https://github.com/davidg327/RemoteJobs.git

Si se descarga por SSH hacer git clone git@github.com:davidg327/RemoteJobs.git


## Una vez este descargado

1. Instalar dependencias

   ```bash
   npm install
   ```

2. Correr la aplicación

   ```bash
   npx expo start y escanear código con la aplicación expo go
   ```

3. Correr los test

   ```bash
   npm run test:coverage
   ```
